# Population Genetics Coach v0.4.0

## Main corrections

- Coordinator workspace hidden from guest and student accounts.
- Direct coordinator URL protected by a role guard.
- Coordinator access allowed only for `coordinator`, `manager`, and `admin` roles.
- Supabase sign-in, registration, sign-out, session restoration, and profile-role lookup added.
- Workspace stage layout rebuilt to prevent text and control overlap.
- Responsive workspace rules added for desktop, tablet, and mobile layouts.
- Lesson content expanded with guided reading, notation checks, interpretation checkpoints, and active-learning consolidation.
- Every lesson displays its approved source basis.
- Learning-path, workspace, and AI interfaces explicitly prohibit web-derived course content.
- AI prompts instruct the evaluator to flag unsupported claims rather than supplement them externally.

## Source boundary

Lesson facts, equations, model notation, examples, and interpretations are restricted to the uploaded QBio305 lectures, exercises, practicals, project guidance, sample projects used only for structural inspiration, and uploaded approved textbooks. Raw teaching files are not included in the application repository.
