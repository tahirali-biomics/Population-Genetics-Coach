**Population Genetics Coach** is an interactive, AI-assisted course companion for university-level population and quantitative genetics. It supports lecture-grounded learning, formative assessment, scientific reasoning, research-project development, and progress tracking within the QBio305 teaching environment.

## Core features

- Eight lecture-grounded population-genetics modules
- Two focused theory extensions
- Guided practical, analysis, writing, presentation, feedback, and exam workspaces
- Course-grounded AI tutor and formative evaluator
- Objective self-assessment and concept-mastery tracking
- Voice input and read-aloud support
- Persistent learner progress
- Role-controlled coordinator analytics
- Responsive access on desktop, tablet, and smartphone

## Academic purpose

The application is designed to help students move from equations and population-genetic models to defensible biological interpretation. It emphasizes assumptions, alternative explanations, limitations, and the distinction between an observed pattern and the evolutionary process that may have generated it.

The AI components are intended for formative support. They do not replace instructor judgement, primary scientific literature, or independent scientific reasoning.

## Source and content boundary

The deployed browser application receives compiled frontend assets only. AI-provider credentials and server-side processing remain outside the frontend bundle.
